# ai-core

Здесь живут docker/compose стеки и конфиги для `vm-ai-core-01` (OpenClaw, автоматизации).

## Можно в git

- `docker-compose.yml`
- `*.env.example`
- манифесты systemd (без секретов)
- скрипты деплоя/проверок

## Нельзя в git

- реальные `.env` с токенами
- приватные ключи, пароли, доступы к почте/API
